# Bioinfo_UERJ

## Materiais introdutórios para alunos de Bioinformática

Bem-vindo(a) ao repositório da disciplina/oficina de Bioinformática da UERJ.
Aqui você encontrará arquivos, notebooks e exemplos práticos para iniciar seus estudos em Python, R, análise de dados biológicos e ferramentas essenciais da área.

Este repositório foi criado para apoiar estudantes que estão começando na área e precisam de um ponto de partida organizado, simples e direto.

## Este material serve como:

* apoio para aulas práticas
* referência para estudos individuais
* base para projetos iniciais em bioinformática
* repositório de exemplos que você poderá adaptar em pesquisas futuras
